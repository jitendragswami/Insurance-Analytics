# Insurance Analytics

## Project Overview
This project analyzes insurance policy data to generate business insights using Excel, SQL, Power BI, and Tableau.

## Tools Used
- Excel
- SQL (MySQL)
- Power BI
- Tableau

## Key KPIs
- Total Policies
- Total Customers
- Age Bucket Wise Policy Count
- Gender Wise Policy Count
- Policy Type Wise Analysis
- Policies Expiring This Year
- Premium Growth Rate
- Claim Status Analysis
- Payment Status Analysis
- Total Claim Amount

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- Dashboard Development
- SQL Querying
- Business Intelligence Reporting

## Project Outcome
Developed an interactive dashboard to help stakeholders monitor policy performance, customer demographics, claims, and premium trends.
